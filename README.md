# ☁️ YouCloud v2.0

**YouCloud** — это инструмент для бесконечного облачного хранения данных, который превращает абсолютно любой файл (архивы, документы, софт) в видеофайл. Полученный ролик можно загрузить на YouTube, используя хостинг как бесплатный жесткий диск без ограничений по объёму!

---

## 🚀 Что нового в версии 2.0

Я полностью переработал софт и добавили функции, которые делают процесс кодирования и автоматизации бесшовным.

### 1. 🔥 Новый стандарт стеганографии: YTV2
* **Меньше пиксели:** Размер кодирующего пикселя уменьшен, что кардинально повысило плотность данных.
* **Выше FPS:** Оптимизированная частота кадров для ускорения воспроизведения и чтения.
* **Максимальная полезная нагрузка:** Теперь в один кадр помещается в разы больше байт информации.

### 2. 🤖 Полная автоматизация YouTube
* **Прямая загрузка:** Забудьте о браузере. Инструмент автоматически загрузит готовое видео на ваш YouTube-канал сразу после завершения кодирования.
* **Интеграция с yt-dlp:** Добавлен встроенный модуль скачивания видео обратно из YouTube для последующего извлечения файлов.

### 3. 🔄 Обратная совместимость
* **Универсальный декодер:** Декодер версии 2.0 абсолютно всеяден. Он без проблем распознает и распакует как старые архивы стандарта **YTV1**, так и сверхплотные файлы нового стандарта **YTV2**.
